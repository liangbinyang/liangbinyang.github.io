关于我学习Java的原因

发表于 2022-01-22 更新于 2022-01-23 分类于 学习Java的一些感受分享

热爱java

要说当初爱上Java语言还得从大二说起，那是我初识Java的时光…
当我第一天学习面向对象的语言时候，我还是云里雾里的，傻傻分不清java语言设计的思想…
什么继承、多态、抽象类…
什么多线程、IO流、文件流…
那时候觉得Java为什么会设计的这么复杂，赶紧跑去学习我的JavaScript去了，默默地在心里对自己说：JavaWScript才是世界上最好的语言嘛…
渐渐的来到大二的暑假…那时候无意中看到一B站里面的一套“全套java实战课程：从入门到精通”…
那时候心里想，这视频这么全的？？？抱着试一试的心态就去尝试看了一周，那一周真的是让我逐渐爱上了这门语言，突然发现原来Java才是世界上最靠谱的语言嘛….hahahaha…然后就死缠烂打似的一个劲学习java基础…

学完java基础后我又一次开始懵逼了…

我在想一个问题，难打就要用java基础里面的JUI图形化界面做系统？？？？(因为当时还没有学习JavaWeb)天哪，难道这玩意（指的是那个Java基础阶段的UI界面）也能开发….布局很潦草，设计很粗糙…

后续….有时间再更新…
